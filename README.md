### How to run
  - Install node_modules
```
npm install
```
  - Laravel Migrate (Method 1)
```
php artisan migrate
```
  - Laravel Seeder (Method 1)
```
php artisan db:seed --class=BorrowSeeder
```
  - Import Database (Method 2)
```
import file library.sql
```
  - User Admin (Method 2)
```
email: admin@admin.com
password: A12345678
```
  - Run Laravel
```
php artisan serve
```
  - Run React + Vite
```
npm run dev
```
